## Quality Gates

### Pre-push (every commit)
```bash
bash .claude/scripts/pre-push-check.sh
```

### Pre-release (before v4.0 tag)
```bash
bash .claude/scripts/release-checklist.sh
```

### Post-release
- Verify all published artifacts (Maven, npm, pub.dev, SPM)
- Verify store apps updated
- Verify website shows new version
- Smoke test: fresh project with v4.0 dependency compiles and runs
