Introducing SceneView 3.0 — 3D & AR as Jetpack Compose composables.

After years of building the bridge between Android and 3D/AR, SceneView 3.0 is a complete rewrite around one idea: nodes are composables.

Scene { } for 3D. ARScene { } for AR. That's it.

What's new:

→ Declarative 3D — ModelNode, LightNode, ViewNode live inside Compose like any other composable
→ State drives the scene — no imperative calls, no lifecycle headaches
→ AR made simple — plane detection, anchors, and tap-to-place in a few lines
→ Compose UI in 3D — embed Card, Text, Button directly in your AR scene with ViewNode
→ Google Filament rendering + ARCore integration under the hood
→ MCP server included for AI-assisted development with Claude, Cursor, etc.

Scene(modifier = Modifier.fillMaxSize()) {
    ModelNode(
        rememberModelInstance(modelLoader, "model.glb"),
        scaleToUnits = 1.0f,
        autoAnimate = true
    )
}

11 lines. Full 3D viewer. Orbit, zoom, animate.
