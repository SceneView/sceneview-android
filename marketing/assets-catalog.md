### How to use in SceneView

```kotlin
val environment = rememberEnvironment(rememberEnvironmentLoader(engine)) {
    createHDREnvironment("environments/studio_small_09_2k.hdr")!!
}

Scene(
    modifier = Modifier.fillMaxSize(),
    environment = environment
) {
    // Your nodes here
}
```

Place `.hdr` files in `src/main/assets/environments/`.

## Recommended asset combos for marketing screenshots

### 1. Product Showcase (e-commerce hero)
- **Model:** ChronographWatch or MaterialsVariantsShoe
- **Environment:** Studio Small 09 (clean studio lighting)
- **Features shown:** PBR rendering, orbit camera, reflections

### 2. AR Furniture Placement
- **Model:** Kenney Furniture Kit (sofa, chair)
- **Environment:** Real camera feed (AR)
- **Features shown:** ARScene, AnchorNode, plane detection, gestures

### 3. Dramatic Hero Render
- **Model:** DamagedHelmet or DragonAttenuation
- **Environment:** Venice Sunset or Industrial Sunset
- **Features shown:** HDR lighting, PBR materials, post-processing (bloom)

### 4. Character Animation
- **Model:** Fox or BrainStem
- **Environment:** Symmetrical Garden
- **Features shown:** autoAnimate, animation controls, natural lighting

### 5. Night Scene
- **Model:** Lantern (emissive)
- **Environment:** Moonless Golf
- **Features shown:** Emissive materials, DynamicSkyNode, FogNode

### 6. Physics Demo
- **Model:** Any small GLB (or use SphereNode/CubeNode primitives)
- **Environment:** Lebombo
- **Features shown:** PhysicsNode, collision, gravity, tap interaction

## Converting assets

### HDR to KTX (for faster loading)

Pre-process HDR to KTX1 using Filament's `cmgen`:

```bash
# Install cmgen (ships with Filament SDK)
cmgen --format=ktx --size=256 environment.hdr
```

KTX loads faster than HDR at runtime. Use HDR for development, KTX for production.

### Optimizing GLB file size

Use [gltf-transform](https://gltf-transform.donmccurdy.com/):

```bash
npx @gltf-transform/cli optimize input.glb output.glb
npx @gltf-transform/cli draco input.glb output.glb  # Draco compression
```

Target: under 5 MB per model for mobile. Under 2 MB for models in production apps.
