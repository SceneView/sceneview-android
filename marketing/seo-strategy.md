**Docs / Getting Started**
> Get started with SceneView in 5 minutes. Add the Gradle dependency, drop a GLB model in assets, write 10 lines of Compose. Full API reference included.

**API Reference (llms.txt)**
> Complete SceneView API reference: all node types, composable signatures, resource loading patterns, threading rules. Optimised for AI coding assistants (Claude, Copilot, Gemini).

**Samples page**
> 14 SceneView sample apps: 3D model viewer, AR tap-to-place, physics demo, text labels, post-processing, cloud anchors, and more. Each with full source code on GitHub.

**GitHub README**
> SceneView — cross-platform 3D & AR SDK for Android and iOS. Jetpack Compose + Filament + ARCore on Android. SwiftUI + RealityKit + ARKit on iOS. 26+ node types. No boilerplate. AI-first design. Apache 2.0 license.

## Backlink Strategy

### Tier 1 — High authority, zero cost to submit

| Site | Submission method | Audience size | Priority |
|---|---|---|---|
| **AndroidWeekly** (androidweekly.net/submit) | Submit article URL via web form | 30 000 subscribers | Critical |
| **Kotlin Weekly** (kotlinweekly.net) | Submit via GitHub issue | 15 000 subscribers | High |
| **dev.to** | Cross-post every Medium article (canonical tag → Medium) | Large dev community | High |
| **Hacker News** (news.ycombinator.com/submit) | "Show HN: SceneView — 3D/AR as Jetpack Compose composables" | 5M+ monthly | High |
| **Reddit r/androiddev** | Text post with article link, lead with value | 300 000 members | High |
| **Reddit r/Kotlin** | Share Kotlin-specific articles | 80 000 members | Medium |
| **Reddit r/augmentedreality** | Share AR-focused articles | 100 000 members | Medium |

### Tier 2 — Relationship-driven

| Site | Action | Timeline |
|---|---|---|
| **Google Dev Library** (devlibrary.withgoogle.com) | Submit SceneView as a featured library | Month 2 |
| **Jetpack Compose community** (Slack / AndroidDev Discord) | Share articles in #compose channel | Ongoing |
| **DroidCon / KotlinConf** | Submit talk proposal; conference proceedings link back | Q3 2026 |
| **ProAndroidDev** (Medium publication) | Pitch article as guest contributor | Month 3 |
| **Codemagic / Bitrise blogs** | Pitch "CI/CD for 3D Android apps" collaboration | Month 4 |

### Tier 3 — SEO link-building

- **Stack Overflow**: Answer questions tagged `arcore`, `filament`, `jetpack-compose-3d` and link to relevant docs/articles where genuinely useful (not spam).
- **GitHub Awesome lists**: Submit to `awesome-android`, `awesome-kotlin`, `awesome-arcore` lists.
- **YouTube description links**: When video assets are published, every video links back to the article and GitHub.

## Content Freshness

Google rewards recently updated content. Action items:

1. Add a `Last updated: YYYY-MM-DD` line to every article.
2. Bump the version number in code samples when releasing 3.3.0, 3.4.0.
3. Publish a "What's new in SceneView X.Y" article with each minor release — reuses existing keyword rankings.
