## 1. Smart Link (sceneview.github.io/go)

A single URL that detects the platform and redirects:

| Platform | Action |
|---|---|
| Android | → Play Store SceneView Demo |
| iOS | → App Store SceneView Demo |
| Desktop browser | → Web demo (model-viewer) |
| Claude/LLM | → MCP setup instructions |

**Shareable links for AI-generated scenes:**
`sceneview.github.io/scene?code=BASE64_ENCODED_SCENE`

When someone shares a SceneView scene generated by Claude:
1. Open on phone → download the app, scene is pre-loaded
2. Open on desktop → renders in browser via model-viewer/Filament.js
3. Open in Claude → shows the code + 3D preview

**Implementation:** Static HTML page with JS platform detection. ~1 hour.

---

## 2. 3D Preview Service

`sceneview.github.io/preview` — a web page that renders 3D scenes from URL parameters.

**Use cases:**
- Claude generates SceneView code → includes a preview link in the response
- Architects share 3D model links → renders in any browser
- Data scientists visualize 3D charts → shareable links

**URL format:**
```
sceneview.github.io/preview?model=URL_TO_GLB
sceneview.github.io/preview?scene=BASE64_SCENE_JSON
```

**Tech:** model-viewer web component (already used on homepage).

---

## 3. Claude Plugin for 3D Artifacts

Enhance the MCP server so Claude can:
1. Generate SceneView code
2. Return a preview link: `sceneview.github.io/preview?model=...`
3. The link shows the 3D model interactively in the browser

**For Claude Desktop specifically:**
- MCP tool `render_3d_preview` that returns an HTML snippet
- Claude shows it as a rich response (like code artifacts)

**Monetization:** Free for basic models, Pro tier for:
- Custom environments (HDR lighting)
- Animation playback
- AR preview ("View in AR" button)
- High-res rendering
- Private model hosting

## 4. Monetized AI 3D App

**Concept:** An app where users describe what they want in natural language, and AI generates a 3D scene.

**Name ideas:** SceneView AI, 3D Creator, AR Builder

**How it works:**
1. User types "a blue sports car on a mountain road at sunset"
2. App calls Claude API with SceneView MCP context
3. Claude generates the scene code
4. App renders it in real-time with SceneView
5. User can edit, share, or place in AR

**Revenue:** Freemium (5 scenes/day free, unlimited = $4.99/month)

**Why now:** No one is doing AI → 3D on mobile with native rendering quality.

## 7. Plugin Strategy — Multiple Small > One Big

### Why multiple small plugins:
- Each plugin appears in MCP registry/marketplace → more visibility
- Specialized plugins rank better in search
- Lower risk — if one fails, others continue
- Faster to ship

### Plugin lineup:
| Plugin | What it does | Monetization |
|---|---|---|
| `sceneview-mcp` | Full SDK reference (exists) | Free |
| `3d-preview-mcp` | Generate 3D preview links | Freemium |
| `ar-builder-mcp` | AR scene builder for mobile | Freemium |
| `model-optimizer-mcp` | Optimize glTF/USDZ models | Pay-per-use |
| `scene-generator-mcp` | AI text → 3D scene code | Pay-per-use |
