# LinkedIn Post Draft — SceneView v3.4

## Option A: Technical (developer audience)

---

SceneView is now on 9 platforms.

What started as an Android-only 3D/AR library is now a cross-platform SDK:
Android, iOS, Web, Desktop, TV, Flutter, React Native.

Same idea everywhere: declare your 3D scene like you declare your UI.

What's new in v3.4:
- Web rendering with Filament.js (same engine as Android, in the browser)
- WebXR AR/VR support
- Desktop with Compose Desktop
- Material 3 demo app with 14 interactive samples
- MCP server on the official registry — AI assistants can now generate correct SceneView code

Try the 3D preview: sceneview.github.io/preview?model=YOUR_GLB_URL

GitHub: github.com/SceneView/sceneview
Website: sceneview.github.io

---

I asked Claude to build me a 3D app. It worked on the first try.

But we went further: the MCP server can now generate interactive 3D preview links. Claude writes the code, and you can see the result in your browser — with AR support on mobile.
