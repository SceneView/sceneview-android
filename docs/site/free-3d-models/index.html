    Usage in SceneView¶

    Android (Jetpack Compose)¶

    // Load from URL (Khronos models — no auth needed)
val modelInstance = rememberModelInstance(
    modelLoader,
    "https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/MaterialsVariantsShoe/glTF-Binary/MaterialsVariantsShoe.glb"
)

// Load from assets folder (bundled models)
val modelInstance = rememberModelInstance(
    modelLoader,
    "models/apartment.glb"  // from app/src/main/assets/models/
)

    iOS (SwiftUI)¶

    SceneView {
    ModelNode(
        url: URL(string: "https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/MaterialsVariantsShoe/glTF-Binary/MaterialsVariantsShoe.glb")!
    )
}

    Web¶

    SceneView.modelViewer("canvas",
    "https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/MaterialsVariantsShoe/glTF-Binary/MaterialsVariantsShoe.glb"
);

    License Quick Reference¶

    License Attribution? Commercial use? Modify?
    CC0 No Yes Yes
    CC-BY Yes (credit author) Yes Yes
    CC-BY-SA Yes + share alike Yes Yes (same license)
    Free Standard (Sketchfab) Check per model Usually yes Check per model
    NASA Public Domain No Yes Yes
    Smithsonian CC0 No Yes Yes

    Rule of thumb: CC0, NASA, and Smithsonian models are safest for commercial demos. CC-BY models require attribution (add to app credits screen).

    Tips for Choosing Models¶

    1. For mobile AR: Keep models under 5 MB. Fox (159 KB), Duck (118 KB), and Cybertruck (9.4K triangles) are ideal.
    2. For product demos: Use MaterialsVariantsShoe or ChronographWatch — they show configurator features (color/material switching).
    3. For investor pitches: Modern Apartment + car configurator + medical anatomy = three massive markets in one demo.
    4. For enterprise demos: Architecture (real estate), automotive (dealerships), and medical (health tech) models carry the most weight.
    5. For technical showcases: Khronos models demonstrate advanced PBR features (transmission, sheen, clearcoat, iridescence).
    6. Always check licenses on Sketchfab models — they can change. Khronos, NASA, and Smithsonian are permanently free.
