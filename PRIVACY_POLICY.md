# Privacy Policy — SceneView Demo

**Last updated:** 2026-03-21

## Overview

SceneView Demo is an open-source demonstration app for the SceneView Android SDK.
It is designed to showcase 3D and AR capabilities and does **not** collect, store,
or transmit any personal data.

## Data Collection

This app does **not** collect:
- Personal information (name, email, phone number, etc.)
- Location data
- Usage analytics or telemetry
- Device identifiers
- Crash reports

## Camera and Sensors

When using AR (Augmented Reality) features, the app accesses your device's camera
and motion sensors. This data is processed **entirely on-device** by ARCore and is
**never** transmitted to any server.

## Third-Party Services

This app uses the following third-party services:
- **Google ARCore**: processes camera and sensor data on-device for AR functionality.
  See [Google's Privacy Policy](https://policies.google.com/privacy).
- **Google Play In-App Updates**: checks for available updates via the Play Store.
  No additional data is collected.

## Open Source

SceneView Demo is fully open source. You can inspect the complete source code at:
https://github.com/SceneView/sceneview-android

## Contact

If you have questions about this privacy policy, please open an issue at:
https://github.com/SceneView/sceneview-android/issues

## Changes

Any changes to this privacy policy will be reflected in this file and in the app's
Play Store listing.
